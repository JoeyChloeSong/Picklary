# Picklary Clip Lite 마스터 디자인 고정 규칙

## 공식 기준

- 기준 이미지: `docs/MASTER_DESIGN_REFERENCE.png`
- 디자인 ID: `clip-lite-dark-hero-screenshot-2026-07-14`
- 기준 화면 크기: 1729 × 1890 px

## 유지해야 하는 핵심 구조

1. 좌측 상단 Picklary Clip Lite 로고와 우측 개인정보 보호 pill
2. 중앙 정렬 영문 eyebrow, 2줄 대형 한글 헤드라인, 한 줄 설명
3. 둥근 외곽 편집기 셸
   - 좌측: 큰 영상 스테이지
   - 우측: 영상 불러오기와 추출하기 2단계 카드
4. 영상 미선택 상태의 중앙 업로드 아이콘과 파일 형식 문구
5. 편집기 아래의 3개 가치 카드
6. 좌측 저작권, 우측 오픈소스 고지 푸터

## 색상 및 시각 언어

- 배경: 짙은 네이비와 블랙의 방사형 그라데이션
- 강조색: 민트 `#64e3bb`, 블루 `#6ea8ff`, 진한 블루 `#2b79ff`
- 카드: 반투명 네이비 패널과 얇은 흰색 테두리
- 제목: 흰색에서 블루·민트로 이어지는 그라데이션
- 모서리: 큰 편집기 28px, 영상 20px, 카드 18px 전후

## 개발 규칙

- `src/styles.css`의 `FUNCTIONAL EXTENSIONS BEGIN` 표시 위는 마스터 디자인 영역입니다.
- 기능 추가 시 새 CSS는 표시 아래에만 추가합니다.
- 기존 디자인 값을 바꿀 필요가 있으면 사용자 승인을 먼저 받습니다.
- `tests/design-master.test.js`가 마스터 CSS 해시와 핵심 구조를 검사합니다.
