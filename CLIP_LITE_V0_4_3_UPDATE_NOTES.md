# Picklary Clip Lite v0.4.3

## 스크린샷 마스터 디자인 고정

- 사용자가 지정한 1729 × 1890 스크린샷을 공식 마스터 디자인으로 등록
- 기준 이미지를 `docs/MASTER_DESIGN_REFERENCE.png`에 포함
- HTML에 디자인 ID 메타데이터 추가
- 실행 시 `<html>`에 동일한 디자인 ID를 기록
- 스크린샷과 일치하는 기존 마스터 CSS를 변경 없이 유지
- 향후 기능 CSS가 마스터 디자인을 덮어쓰지 않도록 확장 영역을 명시적으로 분리
- 마스터 CSS SHA-256 및 핵심 화면 구조를 검사하는 회귀 테스트 추가

## 유지된 기능

- 원본 영상 직접 연결을 통한 빠른 로딩
- 수동 호환 미리보기 변환
- 오디오 음소거 해제 및 볼륨 복원
- `-5`, `-3`, `-1`, 재생, `+1`, `+3`, `+5` 이동 버튼
- I/O 구간 지정과 세 가지 추출 방식
