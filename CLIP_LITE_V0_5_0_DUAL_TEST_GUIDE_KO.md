# Picklary Clip Lite v0.5.3 두 버전 비교 테스트

## 제공 버전

### A. Web Test
- 향후 Netlify 및 picklary.com 공개 후보
- 설치와 업로드 없이 브라우저 안에서 실행
- 브라우저가 지원하는 H.264/AAC 등은 원본 직접 재생
- HEVC/H.265 등은 브라우저에 따라 호환 미리보기가 필요

### B. Windows Native Test
- 개발자 모드와 동일한 계열의 로컬 파일 직접 재생
- Windows 미디어 엔진과 설치된 코덱 사용
- 대용량·HEVC 영상의 즉시 재생 가능성을 비교하기 위한 버전

## 동일 영상으로 확인할 항목

| 항목 | Web Test | Windows Native Test |
|---|---|---|
| 영상 선택 후 길이 표시 시간 | 기록 | 기록 |
| 재생 버튼 후 첫 화면 시간 | 기록 | 기록 |
| 영상 화면 | 정상/검정/호환 안내 | 정상/오류 |
| 소리 | 정상/없음 | 정상/없음 |
| ±1/±3/±5 이동 | 확인 | 확인 |
| I/O 구간 등록 | 확인 | 확인 |
| 내보내기 | 확인 | 확인 |

## 공개 결정 기준
- Web Test에서 사용자의 일반적인 영상이 바로 재생되면 `/clip-lite/`에 공개합니다.
- 특정 HEVC 영상만 실패하면 웹에 호환 안내를 유지하고, Windows Native Test는 별도 다운로드 옵션으로 제공할 수 있습니다.
- Netlify에는 Web Test만 직접 실행형으로 배포됩니다. Windows Native는 다운로드 파일로 제공하는 구조가 적절합니다.
