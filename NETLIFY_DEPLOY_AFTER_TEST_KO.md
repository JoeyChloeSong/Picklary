# Picklary Clip Lite v0.5.5 테스트 후 Netlify 배포

1. `Picklary_With_Clip_Lite_v0.5.5_Full_Overwrite.zip`을 풉니다.
2. 내부 `Picklary` 폴더를 기존 GitHub 저장소에 덮어씁니다.
3. 변경 내용을 Commit하고 `main` 브랜치로 Push합니다.
4. Netlify Build command는 `npm run build`, Publish directory는 `dist`입니다.
5. `/clip-lite/`와 `/clip-lite/downloads/Picklary_Clip_Lite_Windows_v0.5.5.zip`을 확인합니다.
