# Paddle Tier List Update

Date: 2026-07-01

## Goal
Add a Picklary Paddle Tier List to the paddles hub to improve engagement, comparison behavior, internal linking, SEO depth, and return visits.

## What changed
- Added a new `Picklary Paddle Tier List` section to `/paddles/` for Korean and English.
- Added a hero CTA button: `티어표 보기 / View tier list`.
- Added S/A/B/C/D tier rows.
- Added model cards inside each tier with links to detail pages.
- Added purpose-based benchmark picks:
  - All-court
  - Control
  - Power
  - Value
  - Beginner-friendly
- Added a visible methodology note so the section reads as editorial guidance, not an ad or absolute ranking.
- Added a gradient `btn--tier` and responsive tier-list design styles.

## Editorial notes
- Tier labels are presented as Picklary editorial fit tiers, not universal product rankings.
- D tier is present structurally, but no current Picklary database model is placed in D tier.
- The section reminds readers to use Paddle Finder and verify price/approval status before buying.
