PickleLevel Windows 실행 방법
=============================

1) 압축을 풉니다.
2) 폴더 안의 아래 파일을 더블클릭합니다.

   00_RUN_SITE_WINDOWS.bat

3) 검은색 명령창이 열리고 빌드 후 브라우저가 자동으로 열립니다.
   자동으로 열리지 않으면 명령창에 표시되는 주소를 브라우저에 직접 입력하세요.

   보통 주소:
   http://127.0.0.1:8787/ko/
   또는
   http://127.0.0.1:8080/ko/

4) 서버를 끄려면 명령창에서 Ctrl + C를 누릅니다.

주의:
- serve.js 파일을 더블클릭하지 마세요. Windows Script Host 오류가 납니다.
- index.html을 직접 더블클릭하지 마세요. 이 사이트는 로컬 서버로 실행해야 이미지와 링크가 정상 작동합니다.
- npm install은 필요하지 않습니다. 이 실행 방식은 Node.js만 사용합니다.

Node.js가 없다는 메시지가 나오면 https://nodejs.org/ 에서 LTS 버전을 설치하세요.
