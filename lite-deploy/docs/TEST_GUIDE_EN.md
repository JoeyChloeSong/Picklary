# Picklary Lite v0.7.5 EN — User Verification Guide

1. Extract the ZIP completely to a new folder.
2. Run `00_RUN_PICKLARY_LITE_WINDOWS.bat`.
3. Open DualCam Composer.
4. Load one video into each card.
5. Confirm that each play button, timeline, and time display remains visible after both videos load.
6. If the page is taller than the window, use the mouse wheel to reach auto-sync, output settings, and compose controls.
7. Confirm the labels: Video 1 (Left/Top in Output) and Video 2 (Right/Bottom in Output).
8. Click either video and verify that Space and arrow keys affect only the active video.
9. Test both audio auto-sync and manual start points.
10. Compose a result and send it directly to Clip Editor.

If a problem remains, save a screenshot and the output of `01_TEST_NATIVE_WINDOWS.bat`.
