# Picklary Lite v0.7.5 사용자 확인 가이드

1. 새 폴더에 ZIP을 완전히 압축 해제합니다.
2. `00_RUN_PICKLARY_LITE_WINDOWS.bat`을 실행합니다.
3. DualCam 합치기로 이동합니다.
4. A 카드와 B 카드에 영상을 각각 불러옵니다.
5. 두 영상이 로드된 뒤에도 각 영상 아래의 재생 버튼, 타임라인, 시간 표시가 보이는지 확인합니다.
6. 페이지가 창보다 길면 마우스 휠로 아래로 스크롤해 자동 정렬, 출력 설정 및 합성 버튼에 접근할 수 있는지 확인합니다.
7. A 제목이 `1번 영상(출력시 좌측/상단)`, B 제목이 `2번 영상(출력시 우측/하단)`인지 확인합니다.
8. A/B 화면 클릭 후 Space와 좌우 방향키가 활성 영상에만 적용되는지 확인합니다.
9. 사운드 자동 정렬과 수동 시작점 기능을 각각 확인합니다.
10. 합성 후 결과를 Clip 편집으로 보내는 기능을 확인합니다.

문제가 남으면 화면 캡처와 `01_TEST_NATIVE_WINDOWS.bat` 결과를 함께 보관해 주세요.
