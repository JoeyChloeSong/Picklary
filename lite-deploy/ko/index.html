v0.7.5에서 수정된 내용

조작부 가림 오류 수정

DualCam 미리보기를 QVideoSink 캔버스로 변경해 영상 표면이 재생 버튼과 타임라인을 덮지 않도록 했습니다.

스크롤 가능한 DualCam

작은 창과 높은 DPI에서도 자동 정렬, 출력 설정, 합성 버튼까지 접근할 수 있습니다.

출력 위치 기반 명칭

1번 영상은 좌측/상단, 2번 영상은 우측/하단으로 명확하게 표시합니다.

사운드 자동 정렬

공통 소리 구간으로 녹화 시간차를 계산하고 적용 전 사용자에게 결과를 보여줍니다.

4K·좌우 기본

DualCam은 Horizontal 배치와 4K 품질로 시작합니다.

GPU 우선 처리

NVIDIA NVENC를 실제 시험한 뒤 사용하며 실패하면 CPU로 자동 전환합니다.
